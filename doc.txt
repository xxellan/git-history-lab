МЕТОД FIND_MAX

Функция: find_max(numbers)
Назначение: Находит максимальное значение в списке чисел

Параметры:
- numbers: список чисел (list)

Возвращаемое значение:
- Максимальное число в списке
- None, если список пустой

Логика работы:
1. Проверяем, что список не пустой
2. Берем первый элемент как начальный максимум
3. Проходим по всем элементам списка
4. Если находим число больше текущего максимума, обновляем максимум
5. Возвращаем найденное максимальное значение
